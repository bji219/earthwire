How does Phase 2 connect to Phase 1?

The sample browser needs to live somewhere — and may or may not feed back into the live data sequencer.

A

Standalone Sample Tool

A separate /samples page — search, preview, assemble kits, export. Completely independent from the Earthwire sequencer. Phase 1 and Phase 2 are two separate tools in the same app.

Flow: Browse → Pick samples → Build kit → Export .aif to OP-1-F
B

Integrated with Live Data

Bird activity data from eBird drives which bird samples play. High species count → trigger a wren call. The Earthwire sequencer controls sample playback in real time, and the sample browser is how you load sounds into it.

Flow: Pick samples → Assign to Earthwire channels → Live data triggers them → Export kit
C

Both — Loose Connection

Standalone sample tool for kit building and export, but samples can also optionally be loaded as trigger sounds for Earthwire channels. Phase 2 is self-contained but has an integration point.

Flow: Build kits independently + optionally assign samples to live sequencer triggers
